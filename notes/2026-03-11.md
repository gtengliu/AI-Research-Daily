## 总体情况

### 趋势简析
**按技术路线**
- 今日主线从“端到端能做”转向“长时稳定可部署”：进度感知、回退重规划、长时记忆、人类反馈后训练成为高频关键词。
- VLA 与神经符号方法加速融合，强调可解释动作原语与组合泛化能力。
- 仿真到真实（Sim2Real）工具链继续成熟，开源项目更强调可复现和工程闭环。

**按应用领域**
- 操作机器人（manipulation）仍为主战场；
- 灵巧手和长时任务是难点高地；
- 上层 Agent 编排能力开始成为“系统级性能”关键因子。

## 文章列举（高质量论文 5 篇，字段已合并）

### 模型架构

#### 00. TiPToP: A Modular Open-Vocabulary Planning System for Robotic Manipulation
- **简称**：TiPToP
- **领域**：通用具身 / 操作
- **论文链接**：https://arxiv.org/abs/2603.09971
- **项目主页**：https://tiptop-robot.github.io
- **评测**：仿真 + 真机
  - 仿真：IsaacSim 设置（文中总计 28 个场景）
  - 真机：DROID（Franka FR3 + 腕部 ZED Mini 双目）；跨本体验证含 UR5e + 腕部 RealSense D435；WidowX AI + 腕部 RealSense D405
- **计算消耗**：方法本身强调 0 机器人训练数据；运行硬件示例包含 NVIDIA L4 / RTX3080 Laptop / RTX4090
- **代码链接**：论文声明开源，仓库以项目主页为准
- **要点**：模块化开词汇规划，把视觉基础模型与 TAMP 联动，实现“自然语言→多步可执行动作”。

#### 01. NS-VLA: Towards Neuro-Symbolic Vision-Language-Action Models
- **简称**：NS-VLA
- **领域**：通用具身
- **论文链接**：https://arxiv.org/abs/2603.09542
- **项目主页**：https://zuzuzzy.github.io/NS-VLA/
- **评测**：仿真
  - benchmark：LIBERO、LIBERO-Plus、CALVIN（含 1-shot / full-demo）
- **计算消耗**：论文正文未清晰披露统一 GPU/时长总表（附录给出训练协议但未形成单行硬件预算）
- **代码链接**：https://github.com/Zuzuzzy/NS-VLA
- **要点**：神经符号动作原语 + 在线 RL，重点提升低数据与扰动下鲁棒性。

### 长时任务与鲁棒性

#### 02. Beyond Short-Horizon: VQ-Memory for Robust Long-Horizon Manipulation in Non-Markovian Simulation Benchmarks
- **简称**：VQ-Memory
- **领域**：长时操作
- **论文链接**：https://arxiv.org/abs/2603.09513
- **项目主页**：https://vqmemory.github.io
- **评测**：仿真
  - benchmark：RuleSafe（20 条锁规则、10 类保险柜，非马尔可夫长时任务）
- **计算消耗**：4×A100；DP3: batch256/200k steps；RDT/CogACT/π0: 30k steps；多任务 π0: batch64/60k steps
- **代码链接**：项目主页已公开（仓库直链以主页为准）
- **要点**：以离散 VQ 记忆替代原始状态历史，显著改善长时任务稳定性与泛化。

#### 03. See, Plan, Rewind: Progress-Aware Vision-Language-Action Models for Robust Robotic Manipulation
- **简称**：SPR-VLA
- **领域**：鲁棒操作
- **论文链接**：https://arxiv.org/abs/2603.09292
- **项目主页**：https://tingjundai.github.io/SPRVLA/
- **评测**：仿真 + 真机
  - 仿真：LIBERO、LIBERO-Plus（Background/Robot/Language/Layout/Light 扰动）
  - 真机：7-DoF 机械臂，外部相机 + 腕部相机；任务含 Pick-up / Tidy up / Push-T
- **计算消耗**：仿真训练 32×A100-80G；真机每任务 4×A100-80G、5k steps
- **代码链接**：项目主页（仓库直链未在正文明确）
- **要点**：显式进度监测 + 失败回退（rewind）形成闭环恢复机制。

### 人在环后训练

#### 04. DexHiL: A Human-in-the-Loop Framework for Vision-Language-Action Model Post-Training in Dexterous Manipulation
- **简称**：DexHiL
- **领域**：灵巧操作
- **论文链接**：https://arxiv.org/abs/2603.09121
- **项目主页**：https://chenzhongxi-sjtu.github.io/dexhil/
- **评测**：真机为主
  - 设备：Franka Research 3 + DexHand021；系统图含 RealSense D455 与 2×D435
- **计算消耗**：Being-H0.5 全参训练 8×H100 / 60k iterations；人在环微调 1×H100
- **代码链接**：项目主页（仓库直链未在正文明确）
- **要点**：人在环纠错采样 + 重加权后训练，在高自由度灵巧操作上提升明显。

## 今日结论
- VLA 赛道已进入“**稳定性与持续执行能力**”竞争阶段：记忆、回退、人在环与结构化推理是四个核心抓手。
- 未来短期最值得跟踪的指标：
  1) 长时任务成功率；
  2) 失败恢复成本；
  3) 实机迁移效率；
  4) 在线迭代速度。
