#### 1) TiPToP: A Modular Open-Vocabulary Planning System for Robotic Manipulation
- **标题**：TiPToP: A Modular Open-Vocabulary Planning System for Robotic Manipulation
- **简称**：TiPToP
- **领域**：Manipulation / 开词汇规划 / VLA接口
- **论文**：https://arxiv.org/abs/2603.09971
- **项目**：https://tiptop-robot.github.io
- **代码**：https://tiptop-robot.github.io（正文写明 project website and code）
- **评测**：正文在 IsaacSim、标准 DROID（Franka FR3+ZED Mini）和外部 DROID 上对比 π0.5-DROID，并给出 UR5e、WidowX AI 跨本体验证；报告成功率与 task progress。
- **算力**：正文未给统一总预算；实现与评测涉及 GPU 并行 cuTAMP/cuRobo，文中出现 RTX 4090/3080 等配置描述。
- **解读**：方法把VLM感知、抓取候选与GPU-TAMP串成可解释流水线，减少机器人专有数据依赖。贡献在于跨本体可迁移；局限是对分割与几何近似敏感。复现建议先复刻DROID评测协议与抓取候选生成。

- **简称**：NS-VLA
- **领域**：神经符号VLA / VLA+在线RL

- **算力**：正文给出训练迭代与在线RL流程细节，但未提供统一GPU卡型/卡数总表，硬件披露不完整。
- **解读**：核心是“符号原语计划+单调约束切换+chunk级在线RL”联合训练，低数据与OOD更稳。局限是工程链条长且依赖原语质量。复现建议先做LIBERO子集与原语转换消融，再接在线阶段。

- **标题**：See, Plan, Rewind: Progress-Aware Vision-Language-Action Models for Robust Robotic Manipulation
- **简称**：SPR-VLA
- **领域**：鲁棒VLA / 失败恢复
- **论文**：https://arxiv.org/abs/2603.09292
- **项目**：https://tingjundai.github.io/SPRVLA/
