#### 5) Object-Centric Residual RL for Zero-Shot Sim-to-Real VLA Enhancement
- **标题**：Object-Centric Residual RL for Zero-Shot Sim-to-Real VLA Enhancement
- **简称**：Object-Centric Residual RL
- **领域**：residual RL / VLA enhancement / zero-shot sim-to-real
- **论文**：https://arxiv.org/abs/2606.18953
- **项目**：https://www.microsoft.com/en-us/research/articles/object-centric-residual-rl/
- **代码**：正文未披露
- **评测**：仿真 benchmark 为 MuJoCo 中 5 个桌面任务：Cube Lift、Pick-and-Place、Stack Cube、Close Drawer、Stand Cup Up，3 seeds，每任务 20 trials；真机为 Franka Research 3 (FR3) + gripper，基座 VLA 为 GR00T-N1.5，30 teleoperation demos/task。视觉输入含 RGB observation、FoundationPose/SAM2 object pose estimator；真实相机、末端夹爪型号和相机位姿数量正文未披露。真实成功率从 8.4/20 提升到 15.2/20。
- **算力**：正文披露 residual actor 单次 GPU forward 约 0.06 ms，FoundationPose tracking 约 18 ms/frame，VLA inference 约 140 ms；训练总 GPU/时长正文未披露。
- **解读**：方法不让 residual policy 看图像，而看 6-DoF object pose、proprioception 和 base VLA action，避开视觉 sim-to-real gap。贡献是纯仿真 TD3 residual 零样本改进真机 FR3；局限是依赖稳定物体位姿估计。复现优先做 pose noise/dropout ablation。

#### 6) Mem-World: Memory-Augmented Action-Conditioned World Models for Persistent Robot Manipulation
- **标题**：Mem-World: Memory-Augmented Action-Conditioned World Models for Persistent Robot Manipulation
- **简称**：Mem-World
- **领域**：robot world model / memory retrieval / policy evaluation and improvement
- **论文**：https://arxiv.org/abs/2606.18960
- **项目**：正文未披露
- **代码**：正文未披露
- **评测**：仿真/生成式 benchmark 基于 DROID world-model replay，34 条 memory-stressing trajectories + 随机 150 条补充评估，20-step/20-second autoregressive rollouts；策略评测覆盖 π0、π0.5 的 5 个长短程 manipulation tasks，并用 200 generated trajectories/task 做 policy improvement。真实平台为 Franka Emika Panda + 2-DoF gripper；相机为 1 个 wrist-view + 2 个 external third-view cameras，共 3 视角。真实 post-training 采 50 episodes/task，长程任务成功率 58% 提升到 72%。

- **解读**：Mem-World 的 W-VMem 用 wrist-view-centered 4D surfel memory 检索与未来动作相关的历史帧，减少腕相机遮挡和来回运动导致的遗忘。贡献是世界模型评测相关性 r=0.97；局限是无显式物理约束。复现要保留三视角 DROID 和 surfel scoring。

#### 7) SC3-Eval: Evaluating Robot Foundation Models via Self-Consistent Video Generation
- **标题**：SC3-Eval: Evaluating Robot Foundation Models via Self-Consistent Video Generation
- **简称**：SC3-Eval
- **领域**：VLA evaluation / action-conditioned video world model / consistency training
- **论文**：https://arxiv.org/abs/2606.18610
- **项目**：https://weichengtseng.github.io/sc3-eval/
- **代码**：正文未披露
- **评测**：仿真 benchmark 不是传统物理仿真，而是 action-conditioned video rollout evaluator；训练于 381 小时真实 table bussing manipulation 数据，场景含 12 类 utensils/trash objects，评估 7 个真实 VLA checkpoints，在 offline/online、in-distribution/OOD split 上对比 Ctrl-World、IRASim、Cosmos-Predict 2.5。真实机器人本体、末端型号和相机位姿数量正文未披露；论文明确多相机观测，含 wrist view 与 third-person view 的 cross-view inpainting。

#### 3) Benchmarking Action Spaces in Reinforcement Learning for Vision-based Robotic Manipulation
- **标题**：Benchmarking Action Spaces in Reinforcement Learning for Vision-based Robotic Manipulation
- **简称**：Action Space Benchmark
- **领域**：vision-based RL / sim-to-real manipulation / action representation
- **论文**：https://arxiv.org/abs/2606.18594
- **项目**：https://youtu.be/MmXEexVRa18
- **代码**：https://github.com/RL-Sim-to-Real/training
- **评测**：仿真 benchmark 为 MuJoCo 中 PandaPickCuboid 与 PandaPushCuboid，PPO 训练 10 independent runs、1024 parallel envs，比较 pose increment、pose velocity、joint position increment、joint velocity。真机为 Franka Emika Panda + gripper，末端腕部安装 1 个 Intel RealSense D405 camera；Pick 每 action space 12 deterministic trials，Push 每 action space 10 trials。Pick 中 joint velocity 与 joint position increment 均 100% 成功，Push 中 joint velocity 推动距离最高且 jerk 最低。
- **算力**：正文未披露 GPU/训练时长；仅披露 1024 parallel simulation environments。
- **解读**：论文系统显示仿真里最快的 Cartesian action 到真机可能最脆，joint velocity 更平滑也更稳。贡献是把 action-space 对比带到视觉输入和真机 sim-to-real；局限是单 cuboid、单相机、任务简单。复现要保留 D405 腕相机与 domain randomization。
