### A. VLA 架构与推理增强

#### 1) AR-VLA: True Autoregressive Action Expert for Vision-Language-Action Models
- **标题**：AR-VLA: True Autoregressive Action Expert for Vision-Language-Action Models
- **简称**：AR-VLA
- **领域**：VLA / 自回归动作建模
- **论文**：https://arxiv.org/abs/2603.10126
- **项目**：正文未披露
- **代码**：正文未披露
- **评测**：仿真 benchmark：SimplerEnv、PushT、ALOHA（文中图表与段落给出），并报告 BridgeData V2 预训练迁移。真机：WidowX 实测。设备配置：WidowX + 夹爪（末端为夹爪执行器）；单外置侧视相机（正文明确）；wrist/head 相机位姿数量未给出。
- **算力**：训练资源：两阶段训练，Phase-1 为 20,000 steps、batch size 1024，约 2 小时/单张 NVIDIA RTX A6000；附录表还给出总训练步数配置（如 30,000/200,000 对应不同设定）。推理开销：正文强调低时延与异步执行优势，但未给单步 latency/FPS/显存。披露状态：正文+附录有训练超参与部分耗时，仓库未披露。
- **信息来源**：PDF正文、附录、arXiv abs
- **解读**：这篇工作的价值在于把动作生成从“每步重置的反应式头”改成持续记忆的真自回归专家，长时序与遮挡场景更稳定。对部署侧也友好：无需重造 VLM 主干，就能通过动作时序建模提升连续控制质量。

#### 2) Cross-Hand Latent Representation for Vision-Language-Action Models
- **标题**：Cross-Hand Latent Representation for Vision-Language-Action Models
- **简称**：XL-VLA
- **领域**：VLA / 跨手型具身迁移
- **论文**：https://arxiv.org/abs/2603.10158
- **项目**：https://xl-vla.github.io
- **代码**：https://github.com/EmptyBlueBox/DexLatent
- **评测**：仿真 benchmark：正文未给统一公开 benchmark 名称（以任务成功率和多手型迁移实验为主）。真机：双臂 xArm7 + 多种灵巧手（Ability/Inspire/X-Hand/Paxini）及 Unitree G1 上验证。设备配置：单 RealSense L515 前视相机（正文注明 D435 未用于 XL-VLA）；机器人与末端配置明确；wrist/head 位姿数量未披露。
- **算力**：训练资源：8×NVIDIA H100（80GB）训练 60K steps，batch size 128。推理开销：正文未给 latency/FPS/显存。披露状态：正文披露训练硬件与主超参；附录补充数据采集与相机设置；仓库已公开。
- **信息来源**：PDF正文、附录、arXiv abs、项目页（xl-vla.github.io）Code按钮→GitHub（EmptyBlueBox/DexLatent）
- **解读**：核心贡献是“跨手型共享潜变量 + 手型特定解码器”，把不同灵巧手动作空间对齐到统一语义层。工程意义很强：新手型接入成本下降，已有示教数据复用效率提高，能缓解每换一只手就要重训的问题。

#### 3) FutureVLA: Joint Visuomotor Prediction for Vision-Language-Action Model
- **标题**：FutureVLA: Joint Visuomotor Prediction for Vision-Language-Action Model
- **简称**：FutureVLA
- **领域**：VLA / 视觉-动作联合预测
- **论文**：https://arxiv.org/abs/2603.10712
- **项目**：https://github.com/xuxiaoxxxx/FutureVLA
- **代码**：https://github.com/xuxiaoxxxx/FutureVLA（可验证：README 标题与论文同名）
- **评测**：仿真 benchmark：LIBERO（含扩展设定 LIBERO-Plus）、SimplerEnv。真机：Franka 多任务验证。设备配置：Franka + Panda Hand（1-DoF）；1 路 eye-on-base Intel RealSense D435i + 1 路 wrist 相机；head 位姿不适用。
- **算力**：训练资源：预训练约 3 天，16×NVIDIA A100；batch size 256，warmup 5,000 steps（正文给出）。推理开销：正文/附录给出推理相关比较与鲁棒性结果，但未统一披露端到端 latency/FPS/显存。披露状态：正文+附录披露较完整训练配置；代码仓库已公开但未补充统一推理账单。
- **信息来源**：PDF正文、附录、arXiv abs、GitHub（xuxiaoxxxx/FutureVLA）
- **解读**：FutureVLA强调“未来视觉状态 + 未来动作”协同建模，能在动作前就把视觉动态纳入决策，改善前瞻性。相比只做动作回归的范式，它在干扰、视角变化和复杂时序任务上更容易保持策略一致性。

#### 4) DepthCache: Depth-Guided Training-Free Visual Token Merging for Vision-Language-Action Model Inference
- **标题**：DepthCache: Depth-Guided Training-Free Visual Token Merging for Vision-Language-Action Model Inference
- **简称**：DepthCache
- **领域**：VLA / 推理加速
- **论文**：https://arxiv.org/abs/2603.10469
- **项目**：正文未披露
- **代码**：正文未披露
- **评测**：仿真 benchmark：LIBERO。真机：PIPER 6-DoF 操作任务。设备配置：PIPER 机械臂 + 夹爪末端；双 RealSense D435；wrist/head 数量未完全明示（但多相机拓扑明确）。
- **算力**：训练资源：训练无新增（training-free 推理方法）。推理开销：实验在单张 NVIDIA RTX 4090（24GB）上完成；正文报告延迟/吞吐改进趋势，但不同模型设定下绝对 latency/FPS 未统一成单表。披露状态：正文披露推理硬件与效果；附录未见额外训练资源（因无需训练）；仓库未披露。
- **信息来源**：PDF正文、arXiv abs
- **解读**：DepthCache抓住部署痛点：不改权重、不重训，直接靠深度引导做视觉 token 合并。对于已有 VLA 系统，这是低风险提速方案，尤其适合算力受限边缘端，把“可用”性能比进一步向实时闭环推进。

### C. 灵巧操作与触觉学习

#### 1) Contact Coverage-Guided Exploration for General-Purpose Dexterous Manipulation
- **标题**：Contact Coverage-Guided Exploration for General-Purpose Dexterous Manipulation
- **简称**：CCGE
- **领域**：Dexterous RL / 探索奖励设计
- **论文**：https://arxiv.org/abs/2603.10971
- **项目**：https://contact-coverage-guided-exploration.github.io
- **代码**：https://github.com/ruoyiqiao/CCGE（可验证：项目页 code 按钮直链）
- **评测**：仿真 benchmark：Isaac Gym 大规模并行灵巧操作任务组。真机：uFactory xArm + 16-DoF LEAP Hand，含跨手型迁移（Allegro）。设备配置：两路 RGB-D（RealSense D4xx 系）做点云融合；机器人与末端明确；固定外参相机位姿数量=2。
- **算力**：训练资源：依赖 Isaac Gym GPU 并行训练，正文未披露具体 GPU 型号/卡数/总时长/总步数统一账单。推理开销：正文聚焦策略表现与迁移，未披露 latency/FPS/显存。披露状态：正文给并行仿真范式与真实硬件组成；附录补充算法细节；仓库已公开。
- **信息来源**：PDF正文、附录、arXiv abs、项目页→GitHub（ruoyiqiao/CCGE，README含论文标题与arXiv回链）
- **解读**：CCGE把“接触覆盖率”做成任务无关探索信号，解决了灵巧操作里最难的稀疏反馈问题。它的泛化点在于不绑定单任务奖励工程，能把探索能力迁移到新物体和新接触模式，落地潜力高。

#### 3) AdaClearGrasp: Learning Adaptive Clearing for Zero-Shot Robust Dexterous Grasping in Densely Cluttered Environments
- **标题**：AdaClearGrasp: Learning Adaptive Clearing for Zero-Shot Robust Dexterous Grasping in Densely Cluttered Environments
